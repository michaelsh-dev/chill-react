# Chill React

Chill React adalah aplikasi web streaming film berbasis React yang dirancang dengan tampilan responsif
untuk perangkat PC/Laptop dan HP/Mobile. Aplikasi ini memiliki fitur login, register, profil pengguna,
serta sistem berlangganan premium.

## Fitur Utama
- Login dan Register pengguna
- Halaman Home dengan daftar film
- Halaman Detail Film dan Video Player
- Halaman Profil Pengguna
- Sistem Premium dan Pembayaran
- Responsive Design (PC & Mobile)

## Teknologi yang Digunakan
- React JS
- Vite
- React Router DOM
- CSS
